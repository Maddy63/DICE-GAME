HOW TO PLAY
👉Its a Two player game,Players has to choose player1 or player2
👉Game starts with the player1, Player1 has to click on ROLL DICE button
👉The dice will roll and displays the number and the same number added to current field
👉The user can roll the dice as many times he want and every time the score is added to     the current field
👉If the user get the dice score as 1 then all the score in the current field set to 0 and    the turn goes to another player
👉If user wants to save the score before getting the dice score 1 he has to click the    Hold button then the current score is added to the player score and the turn goes to     another player
👉Who get the player score as greater than 100 that player will win and the game stops
